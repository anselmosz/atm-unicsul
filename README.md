# Simulador de cash dispenser ATM

Simulador de caixa eletrônico em python para treino de lógica e organização de código

## Objetivo
Aplicar conhecimentos de algoritmos utilizando estruturas de controle de fluxo e estruturas de dados básicas para modelar e simular comportamentos computacionais no desenvolvimento do simulador de cash dispenser de ATM.

## Como rodar o projeto

#### 1 Instalar o Python

Tenha o python instalado para poder executar o arquivo `main.py`

#### 2 Clonar o repositório e acessar o local

```
git clone https://github.com/anselmosz/atm-unicsul

cd <path to>/atm-unicsul
```

#### 3 Execute o arquivo `main.py` através da linha de comando de sua preferência no diretório onde o arquivo está localizado

```
python main.py
```
